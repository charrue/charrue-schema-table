

## 关于
`@charrue/schema-table-next`是一个基于Element Plus的增强型的表格组件，面向配置开发，快速搭建出表格列表页面。支持Vue3.x。

## 安装

``` bash
npm install element-plus @charrue/schema-table-next --save
```
